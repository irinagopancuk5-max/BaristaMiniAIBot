# 🤖 Barista Mini AI Bot

## 📌 Опис
Це мій перший Telegram-бот, створений на Python.  
Бот відповідає на повідомлення користувача та демонструє базову роботу з Telegram Bot API.

---

## ⚙️ Функціонал
- Команда /start
- Відповідь на текстові повідомлення
- Простий чат-бот без AI (базова версія)

---

## 🛠 Технології
- Python 3
- python-telegram-bot
- Telegram Bot API

---

## 🚀 Як запустити

1. Встановити бібліотеку:
```bash
pip install python-telegram-bot
```

2. Додати токен у `bot.py`:
```python
TOKEN = "YOUR_TOKEN"
```

3. Запустити:
```bash
python bot.py
```
